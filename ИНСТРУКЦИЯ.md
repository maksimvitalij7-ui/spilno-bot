# 🤖 Инструкция по запуску Telegram бота

## Что умеет бот

- 🌅 В 9:00 — спрашивает каждого сотрудника «Ты на работе?»
- 📍 Принимает геолокацию и проверяет: в офисе или нет
- ⏰ В 10:30 — напоминает тем, кто не ответил
- 🌆 В 18:00 — просит написать отчёт за день
- 📊 Ты (руководитель) видишь всех в одном месте
- ⬇️ Экспорт в Excel одной командой


---

## Шаг 1 — Создать бота в Telegram (5 минут)

1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Придумай название: например `ДизайнСтудия Учёт`
4. Придумай username (латиницей, с `bot` в конце): например `design_studio_hr_bot`
5. BotFather пришлёт **токен** — длинная строка вида `7123456789:AAFxxx...`
6. Скопируй этот токен

---

## Шаг 2 — Узнать свой Telegram ID

1. Найди в Telegram бота **@userinfobot**
2. Напиши ему `/start`
3. Он ответит твоим `Id` — например `123456789`
4. Это и есть твой ADMIN_ID

---

## Шаг 3 — Настроить config.py

Открой файл `config.py` и заполни:

```python
BOT_TOKEN = "7123456789:AAFxxx..."   # твой токен от BotFather
ADMIN_IDS = [123456789]              # твой Telegram ID

OFFICE_LAT = 48.6194   # широта офиса
OFFICE_LON  = 22.2981  # долгота офиса
```

**Как узнать координаты офиса:**
1. Открой Google Maps
2. Найди офис
3. Кликни правой кнопкой мыши
4. Первая строка — это координаты (скопируй)

---

## Шаг 4 — Запустить бота

### Вариант А: На своём компьютере (для теста)

```bash
# Установить Python если нет: https://python.org
pip install -r requirements.txt
python bot.py
```

Бот будет работать пока открыт терминал.

### Вариант Б: На бесплатном облаке Railway (постоянно)

1. Зайди на **railway.app** (бесплатно)
2. Зарегистрируйся через GitHub
3. Нажми **New Project → Deploy from GitHub**
4. Загрузи файлы бота
5. В разделе Variables добавь `BOT_TOKEN=твой_токен`
6. Готово! Бот работает 24/7 бесплатно

### Вариант В: На VPS сервере

```bash
# Установить зависимости
pip install -r requirements.txt

# Запустить в фоне
nohup python bot.py &

# Или через systemd (надёжнее)
```

---

## Шаг 5 — Подключить сотрудников

1. Дай сотрудникам **username бота** (например `@design_studio_hr_bot`)
2. Каждый пишет боту `/start`
3. Бот их зарегистрирует автоматически
4. Всё! Больше ничего делать не нужно

## Часто задаваемые вопросы

**Q: Бот перестал работать — что делать?**
Перезапусти `python bot.py`. Если на Railway — он перезапускается сам.

**Q: Как добавить нового сотрудника?**
Просто дай ему username бота и скажи написать `/start`.

**Q: Как изменить время уведомлений?**
В файле `bot.py` найди строки с `run_daily` и поменяй `time(9, 0)` и `time(18, 0)`.

**Q: Как сделать второго администратора?**
В `config.py` добавь его Telegram ID в список `ADMIN_IDS`.

**Q: Данные где хранятся?**
В файле `staff_bot.db` рядом с ботом. Это SQLite база — открывается любым SQLite просмотрщиком.
